# Customer Segmentation Analysis (RFM)

In this project, I analyzed an online retail dataset to better understand customer behavior and purchasing patterns.

## Dataset
Online Retail Dataset from Kaggle

## Tools Used
- Python (Pandas, Matplotlib)

## What I Did
- Cleaned and prepared raw transaction data  
- Calculated Recency, Frequency, and Monetary (RFM) metrics  
- Grouped customers based on their purchasing behavior  
- Visualized customer segments using bar charts  

## Key Findings
- Identified a group of highly valuable customers (Best Customers)  
- Observed loyal customers with regular purchase patterns  
- Detected customers who may be at risk due to inactivity  
- Highlighted how segmentation can support marketing decisions
